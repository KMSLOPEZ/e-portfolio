# Portfolio Website Template

A modern, responsive, and professional portfolio website template perfect for developers, designers, and creatives.

## Features

✨ **Modern Design**
- Clean and professional layout
- Gradient color scheme
- Smooth animations and transitions

📱 **Fully Responsive**
- Mobile-first approach
- Works perfectly on all devices
- Hamburger menu for mobile navigation

🎨 **Customizable**
- Easy to personalize with your information
- Modular CSS with CSS variables
- Simple HTML structure

🚀 **Performance**
- Lightweight and fast
- No external dependencies
- Optimized images

## Sections

1. **Navigation Bar** - Sticky navbar with smooth scrolling
2. **Hero Section** - Eye-catching introduction
3. **About Section** - Personal introduction with stats
4. **Projects Section** - Showcase your best work
5. **Skills Section** - Display your technical skills
6. **Contact Section** - Get in touch form and contact info
7. **Footer** - Copyright information

## How to Use

1. **Edit Personal Information**
   - Replace "John Doe" with your name
   - Update email and social links
   - Add your location

2. **Customize Projects**
   - Update project titles and descriptions
   - Add relevant tech stack tags
   - Change project colors

3. **Update Skills**
   - Add/remove skill categories
   - List your technical skills

4. **Style Customization**
   - Modify CSS variables in `styles.css` for colors
   - Adjust fonts, sizes, and spacing as needed

## CSS Variables

Edit these in `styles.css` to change the entire color scheme:

```css
--primary-color: #667eea;
--secondary-color: #764ba2;
--accent-color: #f5576c;
--dark-bg: #0f172a;
--light-bg: #f8f9fa;
--text-dark: #1a202c;
--text-light: #718096;
```

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations
- `README.md` - Documentation

## JavaScript Features

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)

## Customization Tips

1. **Colors**: Change CSS variables for instant theme changes
2. **Content**: Replace all placeholder text with your own
3. **Images**: Add project images to the `.project-image` divs
4. **Fonts**: Import custom fonts from Google Fonts in the HTML head
5. **Icons**: Add Font Awesome or other icon libraries

## Getting Started

1. Open `index.html` in your browser
2. Start customizing the content with your information
3. Modify styles in `styles.css` if needed
4. Deploy to a hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Deployment

### GitHub Pages
1. Create a GitHub repository
2. Push your files
3. Go to Settings > Pages
4. Select main branch as source

### Netlify
1. Drag and drop your project folder
2. Netlify will host it for free

### Vercel
1. Import your GitHub repository
2. Deploy with one click
